# EasyFoodApp
This app was an app that I developed to practice my studies on MVVM, Coroutines, Fragments, etc.

## Technologies
* Kotlin
  - MVVM
  - Kotlin Coroutines
  - Retrofit to consume an API
  - RecyclerView Adapters
  - LiveData
  - ViewModelFactory
  - DiffUtil
* ViewGroup
    * RecyclerView
    * AppBar Layout
    * NestedScrollView
    * ConstraintLayout
    * CoordinatorLayout
    * CardView
    * CollapsingToolbarLayout
    * LinearLayout
    * Navigation 
* UI Components
  - ImageView
  - FloatingActionButton
  - TextView
  - EditText
  - LinearProgressIndicator
* Glide for image loading
* ViewBinding
